# Pup Picks

A Tinder-style dog toy store where you swipe right to add toys to your cart.

## Features

- **Swipe to Shop**: Browse dog toys by swiping right (add to cart) or left (skip)
- **User Accounts**: Create an account to save your preferences
- **Dog Profiles**: Add your dog's info including name, size, activity level, and favorite toy types
- **Shopping Cart**: View and manage items you've added
- **Invite Friends**: Share the app with other dog owners

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 to view the app.

## Tech Stack

- React
- Vite
- localStorage for data persistence
